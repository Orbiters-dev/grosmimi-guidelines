Category A / PPSU素材の安全性
CAT A PPSU素材の安全性
フック
「素材気にしてなかった〜」 「ppsuって知ってる？」
撮影シーン
・自分が使っていた普通のPPマグを持って話す場面 ・スマホで調べている場面（「マイクロプラスチック」などの検索画面や記事） ・グロミミのマグに切り替わる場面 ・子どもが安心して使っている日常シーン
ナレーション例
「正直マグの素材なんて全然気にしてなかったんやけど、ある日たまたま調べたらびっくりして。PP素材って熱が加わるとマイクロプラスチックが溶け出すリスクがあるらしくて、しかも環境省やWHOも注意喚起してるって知って。毎日子どもに使わせてたのに全然知らんかった…。それから素材にこだわったマグに変えたんよね。PPSUはその心配がなくて、耐熱温度もPPの約100℃に対してPPSUは200℃。熱にも強いし安心して使えるから、これにしてよかったって思ってる」
テロップ例
「PP素材：耐熱約100℃ / マイクロプラスチック溶出リスクあり」 「PPSU：耐熱約200℃ / 溶出リスクなし」 「環境省・WHO　マイクロプラスチックのリスクを警告」
キャプション例
「知らんかった自分が怖すぎた」 「PPのマグ使ってる人に届いてほしい」
Category B / 漏れない設計
CAT B 漏れない設計
フック
「このストロー、逆にしても漏れないって知ってた？」
撮影シーン
・マグを思い切り振っても漏れない実演 ・マグをひっくり返しても漏れない実演 ・クロスカットストローのアップ
ナレーション例
「ストローマグって、バッグの中でひっくり返ってたり、子どもが振り回したりして漏れるのが地味にストレスやったんやけど、グロミミのクロスカットストローは逆さにしても振っても全然漏れないんよ。バッグの中も毎回ヒヤヒヤしなくていいし、子どもがどんな姿勢で飲んでても安心。これだけでもう替えられへんってなった」
テロップ例
「クロスカットストロー：逆さにしても漏れない」
キャプション例
「どの姿勢でも漏れない、それだけで神」
Category C / 発達サポート
CAT C 発達サポート
フック
「マグ選びが発達に関係するって知ってた？」

「口腔発達を促すストロー設計、両手で握れるハンドル、自分で操作できる設計で、赤ちゃんの成長をそっとサポートします」
テロップ例
「口腔サポート設計」「微細運動をサポートするハンドル」「自分で操作できる自立設計」

「飲む練習が、成長の練習になってた」
Category D / お手入れ簡単
CAT D お手入れ簡単
フック
「毎日洗うから、洗いやすさは正義」
撮影シーン
・すべてのパーツを分解して並べる場面 ・各パーツをさっと洗う場面 ・数秒で再組み立てする場面 ・食洗機や煮沸してる場面（対応機種の場合）
ナレーション例
「すべてのパーツが取り外せるから隅々まで洗えて、再組み立ても数秒。毎日清潔に使えます」
テロップ例
「全パーツ分解可能」「食洗機対応」「再組み立て数秒」
キャプション例
「パーツ全部外せるの、これだけで買う理由になる」
Category E / 日常使用シーン
CAT E 日常使用シーン
フック
「公園も保育園も外食も、これ一本で全部いけた」
撮影シーン
・公園で子どもが自分で飲んでいる場面 ・保育園の準備でバッグに入れている場面 ・外食中にテーブルに置いてさりげなく映り込む場面 ・旅行・お出かけ先で使っている場面
ナレーション例
「公園でも、保育園でも、外食でも、旅行でも、気づいたらいつもこれをバッグに入れてる。軽くて子どもも自分で持ちやすいから、どこに行くときも迷わずこれ一本。荷物減らしたいときも、これがあれば安心やなってなってる」
テロップ例
「公園 / 保育園 / 外食 / 旅行」 「軽量設計」「どこでも活躍」
キャプション例
「どこ行くときも気づいたらバッグに入れてた」 「結局これが一番使いやすかった」
